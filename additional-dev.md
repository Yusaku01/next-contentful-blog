# 追加実装メモ

- ルーティング整理: `/blog` と `/blog/[slug]`、`/news` と `/news/[slug]` を追加。旧 `/posts/[slug]` は削除。
- データ取得: `lib/api.ts` に notice 用クエリを追加し、記事・お知らせ両方で前後ナビ用の隣接取得を実装。
- UI 調整: 全体フォント/画像サイズ縮小、コンテンツ幅を絞り、カード UI を導入。フッターの Next.js CTA を撤廃。
- トップページ: ヒーローを刷新し、最新のブログ/お知らせプレビューと各導線を追加。
- ブログ詳細/お知らせ詳細: 幅を狭め、前/次ナビを追加。お知らせは画像任意、サムネなしの一覧。
- sitemap: `app/sitemap.ts` でブログ・お知らせ・トップの URL を自動生成。
- メタデータ: `generateMetadata` を各詳細ページで実装し、タイトル/説明/OGP/カノニカルを自動生成。ルートレイアウトで `metadataBase` を設定。
